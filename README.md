# Navyasri
Dynamic Data Visualization dashboard
The Dynamic Data Visualization Dashboard project focuses on creating an interactive platform to analyze and visualize datasets using Python. The system is built using libraries such as Pandas, Plotly, and Streamlit, which allow users to upload datasets, filter data, and generate different types of visualizations dynamically. 

The main goal of the project is to make data analysis simple and interactive by converting raw data into meaningful visual insights. Users can upload CSV files, apply filters, and view visualizations such as bar charts, line charts, scatter plots, histograms, heatmaps, and pie charts. The dashboard updates charts in real time based on user inputs, enabling easy exploratory data analysis. 

The system includes features like data filtering, theme switching (light/dark mode), real-time chart generation, and downloading filtered data. Compared to traditional tools like Tableau or Power BI, this solution is open-source, lightweight, customizable, and cost-effective. 

Overall, the project demonstrates how modern visualization tools can help users better understand complex datasets, support data-driven decision-making, and provide an interactive environment for analysis.
